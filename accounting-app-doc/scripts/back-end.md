### endpoint get(/balance-sheet)

From: [[migrations/balance-sheet]]

Export JSON: *balanceSheet*
```js
{
	[
		account: "Cash",
		total: XX.XX,
		type: "assets"
	],
	[
		account: "Accounts Payable",
		total: XX.XX
		type: "liabilities"
	],
	[
		account: "Earnings",
		total: XX.XX
		type: "equity"
	],
}
```

### endpoint get(/income-statement)

From: [[migrations/income-statement]]

Export JSON: *incomeStatement*

```js
	[
		type: "revenue",
		category: "Sales",

	],
	[
		type: "cost",

		value: 500.00
	],
	[

```

### endpoint get(/form-data)

Get data from: [[accounts]] and [[categories]]

Export JSON: *formData*

```js
{
	[
		accountID: 1
		accountName: "Cash"
	],
	[
		accountID: 2
		accountName: "Bank Account"
	]
},
{
	[
		categoryID: 1
		categoryName: "Sales"
	],
	[
		categoryID: 2
		categoryName: "Cost"
	]
}
```
